# Knights-Travails (骑士巡游)

## 项目说明
这是一个解决“骑士巡游”问题的 JavaScript 实现。给定棋盘上的起始位置和目标位置，该程序会计算出骑士（马）从起点到达终点所需的最少步数，并输出具体的移动路径。骑士的移动规则遵循国际象棋的标准（走“日”字）。

## 使用的知识点
*   **算法**: 广度优先搜索 (BFS) - 用于寻找最短路径。
*   **数据结构**: 队列 , 图 , 树 (用于回溯路径)。

# Knights-Travails

## Project Description
This is a JavaScript implementation solving the "Knight's Travails" problem. Given a starting square and a target square on a chessboard, the program calculates the minimum number of moves a knight needs to get from the start to the end and outputs the exact path. The knight moves according to standard chess rules (L-shape).

## Concepts Used
*   **Algorithms**: Breadth-First Search (BFS) - used for finding the shortest path.
*   **Data Structures**: Queue, Graph, Tree (for path backtracking).
